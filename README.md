# Lustorri — Where Lust Meets Love

A luxury jewelry e-commerce platform built with Next.js.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- GSAP + Framer Motion
- Zustand
- pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
pnpm build
pnpm start
```
